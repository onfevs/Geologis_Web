## Core Principles

1. **Page-Level Justification**: Every page must answer "Why does this page deserve to exist separately?"
2. **Data Defensibility**: Proprietary > Product-derived > UGC > Licensed > Public
3. **URL Discipline**: Subfolders, one page type per directory, human-readable URLs
4. **Intent Completeness**: Fully satisfy informational/comparative/local/transactional intent
5. **Quality at Scale**: 100 excellent pages > 10,000 weak ones

## The 12 Playbooks

Templates · Curation · Conversions · Comparisons · Examples · Locations · Personas · Integrations · Glossary · Translations · Directories · Profiles

## Phase 1–3 (if Feasibility ≥ 65)

1. **Context**: Business, audience, funnel role, conversion goal
2. **Search Opportunity**: Keyword pattern, page count, demand distribution
3. **Page System Design**: URL pattern, data model, template specification, indexation rules

## Quality Gates

Pre-index: Unique value demonstrated · Intent satisfied · No near-duplicates · Canonicals correct

Kill switch triggers: High impressions + low engagement · Thin content warnings · Index bloat · Suppression signals
